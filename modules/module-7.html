C7 — Analyser

Ici le SST relie cause → effet : danger, déclencheur, dommage et conséquence pour l’équipe.

Mécanisme du dommage

Question clé

Qu’est-ce qui peut déclencher l’accident maintenant ?

Réflexe C7

Je relie la source de danger à la conséquence probable, pas à une opinion.

Cas pratiques interactifs (4 environnements)

BTP

Tranchée ouverte, passage piéton

1. Situation : fin de journée, rubalise partielle, pluie.

2. Où est le danger ?

3. Danger : bord non protégé + sol gras.

4. Analyse : un pas de côté suffit à provoquer une chute dans la tranchée.

5. Action : arrêt passage + barriérage rigide + alerte chef de chantier.

Atelier industriel

Redémarrage machine après bourrage

1. Situation : opérateur en appui près d’un organe mobile.

2. Où est le danger ?

3. Danger : zone de coupe accessible.

4. Analyse : déclenchement intempestif = coupure grave main/avant-bras.

5. Action : consignation + capot fermé + validation chef d’équipe.

Bureau

Allée d’évacuation encombrée

1. Situation : cartons + câble, collègue étourdi.

2. Où est le danger ?

3. Danger : obstacle bas dans axe de marche.

4. Analyse : trébuchement + choc crânien possible.

5. Action : dégagement immédiat + signalement RH/manager.

Manutention

Quai humide et palette instable

1. Situation : charge haute, marche arrière, visibilité faible.

2. Où est le danger ?

3. Danger : sol glissant + charge déséquilibrée.

4. Analyse : glissade + écrasement du pied possible.

5. Action : balisage, séchage, manutention à deux.
