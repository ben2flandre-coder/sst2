C5 — PI SST COMPLET

Arbre décisionnel visuel PI : objectif, repérage, conduite, red flags, erreurs, variantes et cas concrets métier.

Chaîne PI

Danger→Protéger→Examiner→Alerter→Secourir

1. Saignement abondant

Objectif : stopper la perte sanguine.

Signes : saignement continu abondant.

Conduite : compression + allonger + alerter.

Red flags : jet pulsatile, pâleur, confusion.

Erreur : retirer la compression.

Scénario : coupure profonde sur machine.

2. Étouffement

Objectif : désobstruer vite.

Signes : toux inefficace, incapacité parler.

Conduite : 5 claques + 5 compressions.

Red flags : cyanose, perte de connaissance.

Erreur : faire boire.

Scénario : cantine, obstruction alimentaire.

3. Malaise

Objectif : éviter aggravation.

Signes : pâleur, sueurs, douleur thorax, confusion.

Conduite : repos, questionner, surveiller, alerter 15.

Red flags : trouble parole, faiblesse bras.

Erreur : laisser seul.

Scénario : salarié diabétique en salle de pause.

4. Brûlure

Objectif : limiter l'extension tissulaire.

Signes : douleur vive, rougeur, phlyctènes.

Conduite : refroidir 15-20 min, protéger, alerter si grave.

Red flags : visage, mains, voie aérienne.

Erreur : appliquer pommade grasse.

Scénario : projection eau bouillante.

5. Traumatisme / douleur

Objectif : prévenir aggravation.

Signes : impotence, déformation, douleur majeure.

Conduite : repos, immobiliser position, alerter.

Red flags : douleur rachis/cou.

Erreur : mobilisation forcée.

Scénario : chute de plain-pied en atelier.

6. Plaie simple

Objectif : éviter infection et aggravation.

Signes : plaie superficielle non hémorragique.

Conduite : nettoyer, protéger, tracer.

Red flags : corps étranger, morsure.

Erreur : retrait objet planté.

Scénario : coupure mineure en cuisine.

7. Inconscience

Objectif : maintenir ventilation.

Signes : inconscience avec respiration.

Conduite : PLS + alerte + surveillance.

Red flags : respiration irrégulière.

Erreur : abandon surveillance.

Scénario : malaise vestiaire.

8. Arrêt cardiaque + DAE

Objectif : perfusion minimale et choc précoce.

Signes : inconscience + absence respiration normale.

Conduite : RCP immédiate + DAE.

Red flags : interruptions prolongées.

Erreur : attendre secours sans agir.

Scénario : effondrement en production.

9. Risque spécifique / cas concret

Objectif : intégrer les risques de l'entreprise.

Signes : chimique, électrique, thermique, circulation.

Conduite : isoler danger, EPI, alerter, secourir.

Red flags : atmosphère toxique, source active.

Erreur : intervention sans protection.

Scénario : projection chimique maintenance.

Variantes public

Adulte : protocole standard PI.

Enfant : force adaptée, communication rassurante.

Nourrisson : gestes dédiés + alerte médicalisée précoce.
