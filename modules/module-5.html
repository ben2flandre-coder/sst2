C5 — PI SST COMPLET

Référentiel d'intervention structuré : Objectif, Signes, Actions, Vigilances, Erreurs, déclinaisons adulte/enfant/nourrisson.

🚨 Urgence

-10% de survie par minute sans RCP/DAE lors d'un arrêt cardiaque. AT/MP France : sinistralité toujours élevée, nécessité d'une intervention SST fiable.

1) Arrêt cardiaque

Objectif : maintenir circulation/oxygénation avant relais.

Signes : inconscience, absence de respiration normale.

Actions : alerte + RCP + DAE.

Vigilances : minimiser interruptions de compressions.

Erreurs : PLS à tort, absence DAE.

Déclinaisons : adulte (30:2), enfant (adaptation force), nourrisson (2 doigts + insufflations si formé).

2) Hémorragie

Objectif : stopper perte sanguine.

Signes : saignement abondant continu.

Actions : compression directe, alerte, surveillance.

Vigilances : auto-protection, maintien compression.

Erreurs : attendre, retirer compression pour vérifier.

Déclinaisons : adulte (pression ferme), enfant (pression adaptée), nourrisson (pression douce continue).

3) Étouffement

Objectif : libérer voies aériennes rapidement.

Signes : toux inefficace, cyanose, impossibilité parler.

Actions : claques dorsales + compressions adaptées.

Vigilances : alerte en parallèle.

Erreurs : faire boire, manœuvres non adaptées nourrisson.

Déclinaisons : adulte/enfant (abdominales), nourrisson (thoraciques uniquement).

4) Chute traumatique

Objectif : éviter aggravation lésionnelle.

Signes : douleur rachidienne, impotence, choc.

Actions : protéger zone, immobiliser, alerter.

Vigilances : ne pas mobiliser inutilement.

Erreurs : relever la victime.

Déclinaisons : adulte (maintien axe tête-cou-tronc), enfant (rassurer + immobilisation douce), nourrisson (manipulation minimale).

5) Malaise

Objectif : prévenir aggravation brutale.

Signes : pâleur, sueurs, douleur thoracique/confusion.

Actions : repos strict, alerte 15/112, surveillance.

Vigilances : réévaluer état régulièrement.

Erreurs : laisser seul, effort physique imposé.

Déclinaisons : adulte (position de confort), enfant (adapter communication), nourrisson (alerte immédiate + observation continue).

🧠 À retenir

Le PI n'est pas linéaire : chaque étape PAP se réévalue en continu selon l'évolution clinique et les risques résiduels.
