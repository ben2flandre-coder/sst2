PAP SST

Support source exploitable dans la plateforme : observation, danger, situation dangereuse, événement déclencheur, dommage, action de prévention et transmission.

Source intégrée
Observation → Danger → Situation dangereuse → Événement déclencheur → Dommage → Action de prévention → Transmission

Lecture métier

  • Observation : ce qui est visible sur le poste, sans interprétation.
  • Danger : source de dommage.
  • Situation dangereuse : exposition réelle d'une personne au danger.
  • Événement déclencheur : ce qui fait basculer vers l'accident.
  • Dommage : conséquence corporelle ou matérielle attendue.
  • Action : mesure de prévention réaliste et immédiate.
  • Transmission : message utile au bon relais.

Exemple atelier

Observation : capot ouvert, main proche de la zone de coupe.

Danger : organe en mouvement.

Situation dangereuse : accès à la zone de coupe pendant le redémarrage.

Événement déclencheur : relance machine pour débloquer un bourrage.

Dommage : coupure grave / amputation.

Action : arrêter, consigner, fermer, rappeler la procédure.

Exemple logistique

Observation : palette instable sur quai humide.

Danger : charge instable + sol glissant.

Situation dangereuse : manœuvre cariste avec visibilité réduite.

Événement déclencheur : recul et désaxage de charge.

Dommage : écrasement / chute / fracture.

Action : baliser, assécher, stabiliser, reporter le flux.
